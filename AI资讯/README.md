# AI 资讯报告目录

本目录保存需要进入 `yuanfang11223344/AI.git` 的 AI 日报和周报。

## 文件类型

| 类型 | 文件名 | 生成条件 |
|---|---|---|
| 日报 | `YYYY-MM-DD.md` | 默认每日生成 |
| 周报 | `weekly-YYYY-WXX.md` | 周一且本周周报不存在时生成 |

## 同步要求

本目录中的报告应与以下路径保持同名同步：

- `/Users/ganxuanzhi/Documents/Obsidian Vault/AI资讯/output/`
