### 🔥 Anthropic 出口管制 + IPO 双线并行

美国商务部 6/12 发布出口管制指南，禁止外国获取 Anthropic 最新模型 Mythos5/Fable5。起因：Amazon 研究人员发现可提取受限网络攻击信息的漏洞，CEO Andy Jassy 直接报告白宫。Anthropic 高层 6/16 会见特朗普政府磋商解除，警告"如果此标准持续，整个行业发布新前沿模型将几乎不可能"。同时 Anthropic 已向 SEC 秘密提交 IPO S-1，估值 ~$9650 亿（5 月封闭融资 $650 亿）。

- [Yahoo Tech: OpenAI 员工撤回法庭之友签名](https://tech.yahoo.com/ai/claude/articles/amicus-ill-pass-anthropic-policy-033022037.html)
- [AJU Press: G7 AI Leaders](https://m.ajupress.com/view/20260616081270615)

### 🔥 Anthropic Claude Fable 5 隐藏限制争议

Claude Fable 5 被发现内置隐藏限制——检测到"前沿 AI 研究"用途时自动降级性能（变相阻碍竞品）。遭研究人员强烈批评后 Anthropic 逆转决定，道歉称"错误的权衡"。OpenAI 员工 Brian Fioca 撤回对 Anthropic 法庭之友签名："我无法真实地论证他们不是供应链风险"。

- [HKET 中文报道](https://inews.hket.com/article/4145952)

### 🔥 NVIDIA 发布 Nemotron 3 Ultra（5500 亿参数开源，Apache 2.0）

MoE 架构，550B 参数。推理速度比同级前沿模型快 5x，成本低 30%。已部署于 CrowdStrike 安全 agent 和 Palantir 自主运营。Hermes Agent / LangChain Deep Agents / OpenClaw / OpenHands / OpenCode 均已适配。HuggingFace / ModelScope / OpenRouter / NVIDIA NIM 可获取。

- [IT之家 中文报道](https://m.ithome.com/html/958090.htm)
- [HuggingFace: Cohere North Mini Code](https://huggingface.co/blog/CohereLabs/introducing-north-mini-code)

### 🔥 OpenAI 启动 $1 万亿+ IPO

高盛 + 摩根士丹利承销，6/8 向 SEC 秘密提交 S-1。同时设立 $400 亿企业部署部门，大量招聘驻场工程师——与 Anthropic、Oracle 的大裁员形成鲜明对比。2026 年截至目前科技行业裁员 14.2 万人。

- [TNW](https://thenextweb.com/news/g7-ai-summit-altman-amodei-hassabis)

### ✈️ SpaceX IPO 融资 $750 亿，市值突破 $2 万亿

## 🖥️ 芯片与互连（SerDes / CPO / 硅光子）

### 🔥 Credo Blue Heron：业界首个 224G 多协议 AI Retimer

Credo 1/29 推出 Blue Heron 224G AI Scale-Up Retimer，业界首个同时支持 UALink、ESUN 和 Ethernet 三协议的单芯片。3nm 工艺，224G PAM4 SerDes，30-Tap FFE + 16 反射消除抽头，40+dB 链路完整恢复。目前送样，Q3 2026 量产。Scale-Up 网络市场预计 2030 年超 $400 亿。

- [Nasdaq 新闻稿](https://www.nasdaq.com/press-release/credo-introduces-industrys-first-224g-multiprotocol-ai-scale-retimer-supporting)
- [TSMC Symposium 展示](https://www.eenewseurope.com/en/credo-to-unveil-ai-interconnect-solutions-at-tsmc-2026-symposium/)
- [BusinessWire](https://secure.businesswire.com/news/home/20260422367858/en/Credo-to-Highlight-Next-Generation-Connectivity-Solutions-for-AI-Infrastructure-at-TSMC-2026-Technology-Symposium)

### 🔥 Lightmatter + Synopsys：224G SerDes 对接共封装光学 (CPO)

Lightmatter 将 Synopsys 硅验证 224G SerDes + UCIe IP（3nm）集成到 Passage 3D CPO 平台。电气→光学域桥接在 224G 速度下实现超低延迟。Synopsys AI-EDA 工具链（3DIC Compiler / Lumerical / OptoCompiler）用于光电联合设计。650 Group 称"CPO 生态成熟的关键一步"。

- [Lightmatter 官方新闻稿](https://lightmatter.co/press-release/lightmatter-collaborates-with-synopsys-to-integrate-advanced-interface-ip-with-its-passage-co-packaged-optics-platform/)
- [QuantumZeitgeist](https://quantumzeitgeist.com/lightmatter-synopsys-ai-infrastructure-photonic-ai-engines/)

### ⭐ TSMC Symposium 2026：224G PAM4 SerDes on 3nm 成为主旋律

Credo 在 TSMC 全球 Symposium 巡展（Santa Clara / Boston / 新竹 / 阿姆斯特丹 / 上海 / 横滨）展示 224G PAM4 SerDes IP on TSMC 3nm，单模块聚合 1.6Tbps。同时展示 OmniConnect Weaver 内存互连方案（112G VSR SerDes），I/O 密度比 LPDDR5X 高 10x。

### ⭐ COMPUTEX 2026：黄仁勋和 Marvell CEO 定调"光学是唯一缩放路径"

黄仁勋："你用光学来 Scale-Up、Scale-Out、Scale-Across。"Marvell CEO Matt Murphy 呼应。224G SerDes + 光互连被视为 AI 扩展的唯一物理路径。

### 📌 PCI-SIG 成立光学工作组——PCIe 7.0/8.0 将支持光学 PCIe

IEEE 802.3dj 正在从 25G 迁移到 100G PCS Lanes，OSFP-1600 成为标准封装。CPO 测试从单通道验证迁移到 256 通道（未来 1024 通道）系统级光学验证。

## 📄 论文速递

### 💡 Agent-to-Agent Theory of Mind (2506.22957)

测试 LLM 之间的心智理论——一个 agent 能否理解另一个 agent 的知识状态。对多 agent 协作系统有直接指导意义。

- [arXiv 2506.22957](https://arxiv.org/abs/2506.22957)

### 🛠 幻觉控制在数学上不可能 (stat.ML)

证明了任何足够强大的语言模型都存在不可消除的幻觉边界——一篇改变模型部署策略的 paper。

### 🧪 推理能力训练会增加工具调用幻觉 (多篇一致结论)

多篇论文 (2506.23840, 2506.12353, 2506.11274) 从不同角度验证：thinking tokens 增加推理能力的同时也增加了工具调用错误率。

- [2506.23840: Do Thinking Tokens Help or Trap?](https://arxiv.org/abs/2506.23840)
- [2506.11274: Learning a Continue-Thinking Token](https://arxiv.org/abs/2506.11274)
- [2506.12353: Efficient Reasoning Through Suppression](https://arxiv.org/abs/2506.12353)

### 🏗 Agent-RLVR：用环境奖励训练 Coding Agent (2506.11425)

RL-based，无需人工标注，直接用环境反馈（编译/测试结果）训练代码 agent。ACL 2025。

- [arXiv 2506.11425](https://arxiv.org/abs/2506.11425)

### 🧬 ClinHallu：阶段诊断医疗 MLLM 幻觉 (2606.14697)

新 benchmark 逐阶段诊断多模态医疗 LLM 的幻觉，ICML 2026 接收。

- [arXiv 2606.14697](https://arxiv.org/abs/2606.14697)

## 💬 观点/讨论

### 🔶 "Tokenmaxxing" 反噬

多家公司开始反思"token 越多 = AI 用得越好"。关键指标从 token 消耗量转向实际生产力提升。TokenTuner 等成本优化工具开始流行。

### 🔷 Anthropic 信任危机继续发酵

Yahoo Tech 报道评论区：OpenAI 员工撤回法庭之友签名引发行业对 Anthropic"隐藏限制"行为的广泛质疑。讨论焦点：AI 公司自查机制是否足够透明。

- [Yahoo Tech](https://tech.yahoo.com/ai/claude/articles/amicus-ill-pass-anthropic-policy-033022037.html)

### 🟠 AI 裁员两极化：Oracle 裁 1 万 vs OpenAI 扩招

Oracle 用省下的 $80-100 亿买 GPU 和数据中心。OpenAI 反向：设立 $400 亿企业部署部门大量招聘。2026 年至今科技行业裁 14.2 万人，但 AI 公司招聘需求同时创新高。
