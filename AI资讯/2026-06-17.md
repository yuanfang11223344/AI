## 🖥️ 芯片与互连

- 🔥 **NVIDIA 把 Spectrum-X Ethernet Photonics 定位为大规模 AI 网络的下一代基础设施。** 官方页面给出的核心信号是 CPO 直接上 ASIC、带宽到 409.6 Tb/s、计划在 2026 年下半年供货，说明硅光 + CPO 已从概念验证走向产品化窗口。来源：[NVIDIA Silicon Photonics Networking for Agentic AI](https://www.nvidia.com/en-us/networking/products/silicon-photonics/)
- 🔥 **NVIDIA 与 Coherent 的战略合作继续强化“光互连是 AI factory 扩展关键路径”的判断。** 如果交换芯片、外置激光源和 CPO 封装能协同成熟，scale-up / scale-out 网络的功耗与布线瓶颈会明显缓解。来源：[NVIDIA and Coherent Partnership](https://nvidianews.nvidia.com/news/nvidia-and-coherent-announce-strategic-partnership-to-develop-optics-technology-to-scale-next-generation-data-center-architecture)
- ⭐ **Marvell 明确把 224G SerDes 作为 scale-up 互连卖点。** 其最新方案强调用 silicon-proven 224G SerDes 直接驱动 cable package attach connectivity，目标是绕开传统 PCB routing 损耗，这和高密度 AI rack 内互连的痛点高度贴合。来源：[Marvell Scale-up Network Solutions](https://www.marvell.com/blogs/scale-up-network-solutions-for-ai-infrastructure.html)
- ⭐ **Coherent 在 OFC 2026 investor deck 中披露了 socketed CPO、外置激光源与高量订单进展。** 对产业链的含义是：CPO 不再只看实验室 demo，良率、laser ramp 和客户预订量正在成为更关键的判断指标。来源：[Coherent OFC 2026 Investor Deck](https://www.coherent.com/content/dam/coherent/site/en/documents/investors/investor-presentations/2026/march-17/OFC-2026-Investor%20event-deck-vf.pdf)
- 📌 **Marvell 继续提醒硅光制造的瓶颈已经转向光学测试吞吐。** 如果测试环节无法从“分钟级、定制化”转向更标准化 ATE 体系，CPO / silicon photonics 的量产爬坡会受限。来源：[Marvell Optical Test](https://www.marvell.com/blogs/why-scale-up-ai-networks-demand-scalable-optical-test.html)

## 📄 论文速递

- ⭐ **Communication Policy Evolution for Proactive LLM Agents**：讨论 proactive agent 在长期交互中如何逐步学习更合适的沟通策略，重点在“任务约束无法一次性完整表达”这一现实问题。来源：[arXiv:2606.14314](https://arxiv.org/abs/2606.14314)
- ⭐ **Decentralized Multi-Agent Systems with Shared Context**：提出 DeLM，把多 agent 协作建立在 shared verified context 与 task queue 上，强调去中心化协调而不是单一控制器调度。来源：[arXiv:2606.10662](https://arxiv.org/abs/2606.10662)
- ⭐ **TrustMargin: Training-Free Arbitration between Parametric Memory and Retrieved Evidence in Large Language Models**：关注 LLM 参数记忆与检索证据之间的冲突仲裁，属于 RAG 可靠性与 hallucination 控制的直接相关工作。来源：[arXiv:2606.08397](https://arxiv.org/abs/2606.08397)
- ⭐ **M*: A Modular, Extensible, Serving System for Multimodal Models**：聚焦 multimodal serving system 的模块化与可扩展性，对多模态模型工程落地比单纯 benchmark 更有参考价值。来源：[arXiv:2606.12688](https://arxiv.org/abs/2606.12688)
- ⭐ **Scaling LLM Reasoning from Minimal Labels: A Semi-Supervised Framework**：核心思路是用极少监督信号扩展 reasoning learning，并把 reasoning verification 本身转成数据生成机制。来源：[arXiv:2606.16811](https://arxiv.org/abs/2606.16811)
- 📌 **Energy-Efficient On-Device RAG on a Mobile NPU: System Design and Benchmark on Snapdragon X Elite**：面向端侧 RAG，重点是系统设计与能效 benchmark，而不是泛化地宣称“端侧一定更好”。来源：[arXiv:2606.11257](https://arxiv.org/abs/2606.11257)
- 📌 **Understanding the Behaviors of Environment-aware Information Seeking Agents**：分析 LLM 如何针对不同 retriever 学习不同 query formulation 策略，和 agent + search tool 的实际效果高度相关。来源：[arXiv:2606.16817](https://arxiv.org/abs/2606.16817)
